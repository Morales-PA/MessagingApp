# MessageApp (Symfony)

Aplicación de mensajería desarrollada con Symfony y MySQL.  
Permite gestionar usuarios, contactos y conversaciones.
Componentes utilizados: Ratchet, doctrine y mailer.
---

## Requisitos:

- PHP 8.1 o superior.
- Crear un proyecto de symfony.
- Clonar el repositorio (https://github.com/Morales-PA/MessagingApp).
- Sobre la ubicacíon de archivo del proyecto ejecutar el comando en cmd: symfony server:start
- Entar a la aplicación web (localhost:8000)
